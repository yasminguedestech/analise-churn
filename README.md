# 📊 Análise de Churn — Por que clientes abandonam um app?

## Objetivo
Identificar o perfil de clientes que cancelam a assinatura de um app,
usando dados e visualizações para apoiar decisões de produto.

## Metodologia
Simulação de base de dados com 300 clientes e análise de 4 variáveis:
- Taxa de churn geral
- Churn por plano de assinatura
- Churn por faixa de uso mensal
- Tempo médio ativo vs inativo

## Principais insights
- Aproximadamente 30% dos clientes se tornaram inativos
- Clientes do plano Basic apresentam maior taxa de churn
- Clientes com menor uso mensal tendem a se tornar inativos
- Clientes mais novos cancelam mais que os antigos

## Conclusão
Tomar decisões baseadas em dados permite identificar padrões
e agir de forma estratégica para reduzir o churn.
